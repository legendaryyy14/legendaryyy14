F L E X Y

Flexy allows a user to look up a workout plan that includes exercises or simply create their own. This web application uses flask_login for session management and authentication, SQLAlchemy for database operations, and WTForms for form validation. It provides routes for login, logout, signup, and authentication checks, ensuring a structured and secure user authentication system within a Flask application. For the frontend, I employed React with Redux (JavaScript) to craft a dynamic and intuitive user interface, enhancing the overall user experience.

AirBnB Clone

This clone was made following AirBnB’s framework allowing the user to book stays in certain locations. This code implements a session-based authentication system using cookies and middleware in Express.js. It ensures secure storage and comparison of passwords using bcrypt, handles input validation to prevent common security vulnerabilities, and manages user sessions via cookies for a seamless authentication experience. I utilized Express (JavaScript) as the backend framework to create a powerful and flexible server-side architecture. I also implemented React with Redux (JavaScript) to construct a dynamic and interactive user interface, mirroring the Airbnb user experience.
